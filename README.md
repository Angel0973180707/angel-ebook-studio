# Angel Ebook Studio (PWA)

- 書庫：多本書、草稿/完稿
- 編輯器：靈感素材 / 草稿 / 完稿 / AI 指派
- AI 協作：狀態 × 角度（預設 ChatGPT，Gemini / Claude 備選）
- 匯出/匯入：每一本書可單獨匯出為 JSON

## 檔名建議（GitHub Repo）
- `angel-ebook-studio`
